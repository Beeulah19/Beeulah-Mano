# Pilabz Heat Pump Dashboard

Static HTML/CSS implementation converted from the supplied Figma frame:

https://www.figma.com/design/GO921yO2HGfNjIgNYl8JdI/RE?node-id=454-10429

## Files

- `index.html` - Pilabz Heat Pump Dashboard case study
- `instrumentation.html` - Milli-Ohm Meter & Power Analyzer case study
- `uva.html` - Pilabz UVA case study
- `styles.css` - shared responsive styling
- `assets/` - downloaded Figma image assets and the reference screenshot

## Preview

Open `index.html` in a browser. Use the top navigation to switch between pages.
No build step or dev server is required.
